# Gradle

```bash
mkdir /usr/local/dev
cd /usr/local/dev
wget https://services.gradle.org/distributions/gradle-2.10-bin.zip
unzip gradle-2.10-bin.zip
ln -s ./gradle-2.10 gradle

echo "export GRADLE_HOME='/usr/local/dev/gradle'" >> ~/.bash_profile
echo "export PATH=$PATH:$GRADLE_HOME/bin" >> ~/.bash_profile
```
